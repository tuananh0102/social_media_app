# Social Media app
   - BackEnd: ASP .NET, SQLServer
   - FrontEnd: HTML, CSS, ReactJs.
## Features:
- Write APIs for functions like CRUD
- Login, register
- Share, comment post
- Search user and see profile
- Add/Accept friend
- Change avatar
